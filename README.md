## Tech Stack
**Server:** Python -> Flask 
## Installation

Install my-project with pip

```bash
  pip install -r requirements.txt 
  cd my-project
```
## run

run my-project with python

```bash
 python src/app.py
```

## important

if you are going to install new dependencies before uploading run the command 

```bash
 pip freeze > requirements.txt
```
